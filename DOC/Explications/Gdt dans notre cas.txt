------- Comment ca marche.


DPL :   Descriptor Privilege Level


Le cpu check le CPL (level du code courrant) et verifie si
l user peut l exec.

// user/hello.c
void _start()
{
    // syscall write hardcodé en ASM inline
    asm volatile (
        "mov $4, %eax\n"   // syscall write
        "mov $1, %ebx\n"   // stdout
        "int $0x80\n"
    );
    while(1);
}
